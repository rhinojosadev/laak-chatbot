El proyecto consiste en el desarrollo de una plataforma de e-commerce personalizada para la venta de productos físicos, utilizando React, Node.js, MongoDB, Express, GitHub y Vercel. El equipo trabajará en la creación de una página web funcional con carrito de compras, gestión de usuarios, pasarela de pagos, panel administrativo y diseño adaptable. Se requiere equipo de cómputo con al menos un procesador Intel Core i5 de décima generación, 8 GB de RAM y sistema operativo compatible con herramientas modernas de desarrollo web como Windows 10, macOS o Ubuntu 20.04. Para acceder al repositorio en GitHub, cada miembro debe tener una cuenta activa, enviar su usuario al Project Manager, solicitar acceso y aceptar la invitación para luego clonar el repositorio localmente. El equipo está conformado por Juan Pérez como Project Manager, María López como frontend lead, Carlos Jiménez como backend lead, Andrea Rivas en testing y QA, Luis Navarro en UI/UX, Paula Estrada en marketing y SEO, Jorge Salgado como gerente general, Valeria Torres como responsable de contenido y Renato Vega como encargado de finanzas y pagos. Las tecnologías utilizadas son React 18, Node.js 20, MongoDB, TailwindCSS y Express. Toda la configuración del entorno está detallada en el archivo README.md del repositorio. Si el frontend no conecta con el backend, se deben revisar las URLs y que el backend esté activo. Los pull requests deben ser revisados por María López o Carlos Jiménez. El presupuesto está disponible en Google Drive, y cualquier solicitud de compra o licencia debe hacerse con Renato Vega. El horario de trabajo es de lunes a viernes de 9:00 a 17:00 h con flexibilidad para trabajo remoto. En caso de emergencias, se debe avisar al Project Manager. Hay recursos de capacitación disponibles y posibilidad de mentoría. Cualquier duda general puede dirigirse a Juan Pérez o a Jorge Salgado.
